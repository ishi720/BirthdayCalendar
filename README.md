# BirthdayCalendar

## 概要

キャラクターの誕生日をカレンダーに登録します

## 環境

```
$ ruby -v
ruby 3.2.3 (2024-01-18 revision 52bb2ac0a6) [x64-mingw-ucrt]

$ rails -v
Rails 7.1.3.2
```

## セットアップ

```bash
# パッケージインストール
$ bundle install

# DBマイグレート
$ rails db:migrate

# 起動
$ rails s
```
